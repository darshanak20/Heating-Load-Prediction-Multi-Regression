## Energy Efficiency Prediction 

This project predicts building heating load using multiple regression techniques.

### 📊 Models Used:
- Linear Regression
- Ridge Regression
- Lasso Regression
- ElasticNet
- Polynomial Regression

### 🧠 Key Highlights:
- Detected multicollinearity using VIF
- Applied regularization techniques
- Compared model performance using R², MAE, RMSE
- Achieved best results with Polynomial Regression (R² = 0.99)

### 🚀 Tech Stack:
Python, Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib
